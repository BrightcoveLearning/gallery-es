Controlar el acceso a una experiencia de portal

{{ page.description }}

Los perfiles de control de acceso se utilizan para controlar el acceso a las experiencias del portal. Los perfiles de control de acceso se crean como parte de la configuración de la Galería y pueden restringir el acceso a un sitio mediante:

  • Limitar el acceso según la dirección IP
  • Requerir que el usuario ingrese credenciales (SSO)
  • Requerir que el usuario ingrese un código de acceso

Para obtener información sobre la creación de perfiles de control de acceso, consulte Configuración de perfiles de control de acceso. Una vez que se han creado los perfiles, se pueden asignar a Portal Experiences para controlar el acceso. Para asignar un perfil de control de acceso a un sitio, siga estos pasos:

  1. Edite su experiencia de portal.
  2. Haga clic en el CONFIGURACIÓN DEL SITIO> Control de acceso en la navegación de la izquierda.
  3. Seleccione un perfil de control de acceso para el sitio. De forma predeterminada, todos los sitios son públicos.
  4. Hacer clic Ahorrar.

Asegúrese de volver a publicar el sitio después de realizar cualquier cambio en la configuración de control de acceso.
