Uso de un formulario de cliente potencial con una experiencia de portal

{{ page.description }}

Los formularios de clientes potenciales se pueden usar para capturar información de los espectadores mientras los espectadores ven videos y se pueden usar con los siguientes tipos de experiencias:

  • Portal
  • Portal de eventos en vivo
  • Página de destino

Con el portal, el portal de eventos en vivo y algunas experiencias de la página de destino, los formularios de clientes potenciales se mostrarán dentro del reproductor. Los formularios de clientes potenciales se pueden configurar para que se muestren después de que se haya visto una cantidad específica de videos o después de que haya pasado un intervalo de tiempo predeterminado. Los pasos para utilizar un formulario de cliente potencial son los siguientes:

  • Cree un formulario de cliente potencial utilizando el módulo Audiencia
  • Asignar el formulario de cliente potencial a un reproductor habilitado para la audiencia
  • Configurar una experiencia para usar el formulario de clientes potenciales
  • Exportar los datos del cliente potencial

Crear un formulario de cliente potencial

Los formularios de clientes potenciales se crean mediante el módulo Audiencia. Para conocer los pasos completos para crear un formulario de cliente potencial, incluidas todas las opciones disponibles, consulte Creación de un formulario de cliente potencial para la audiencia. Para crear un formulario de cliente potencial, siga estos pasos:

  1. Abra el módulo Audiencia.
  2. Hacer clic Formularios de clientes potenciales en la navegación de la izquierda.
  3. Hacer clic Crear formulario de cliente potencial.
  4. Entrar a Nombre para el formulario principal.
  5. Configure los ajustes del formulario de cliente potencial y luego haga clic en Ahorrar.
Nota: No se realiza ninguna validación de los datos del formulario de clientes potenciales que se han introducido.
Nota: Cuando se utiliza un formulario de clientes potenciales en una experiencia de página de destino y la experiencia se configura para mostrar el formulario en el cuerpo de la página, se ignorarán los ajustes de visualización y temporización del formulario de clientes potenciales.

Asignar un formulario de cliente potencial a un reproductor habilitado para la audiencia

Los formularios de clientes potenciales solo se pueden mostrar en reproductores habilitados para público. Un reproductor habilitado para Audience es un Brightcove Player que está configurado con el complemento Audience. Los reproductores habilitados para la audiencia graban datos de participación en el video que se pueden ver dentro del módulo Audiencia. Para crear un reproductor habilitado para público, siga estos pasos:

  1. Abra el módulo Audiencia.
  2. Hacer clic reproductores de la audiencia.
  3. Hacer clic Crear reproductor para crear un nuevo reproductor habilitado para público.
    Nota: También es posible agregar el complemento Audience a un reproductor existente.
  4. Localice su reproductor y utilice el Formulario de cliente potencial lista desplegable para seleccionar el formulario de cliente potencial que se mostrará.
  5. (Opcional) Si está utilizando Audience para realizar un seguimiento de los datos de participación en el video, el reproductor debe configurarse con una conexión de datos mediante el Conectar a la lista desplegable.

Configurar una experiencia para usar un formulario de cliente potencial

Una vez que se ha creado y asignado un formulario de cliente potencial a un reproductor habilitado para Audiencia, se puede usar en una experiencia de portal. Para configurar una experiencia para usar un formulario de cliente potencial, siga estos pasos:

  1. Abra el módulo Galería.
  2. Edite la experiencia a la que desea agregar el formulario de cliente potencial.
  3. Hacer clic VIDEO Y REPRODUCCIÓN> Formulario de reproductor y cliente potencial en la navegación de la izquierda.
  4. Seleccione un reproductor habilitado para audiencia que esté configurado con el formulario de cliente potencial para mostrar.
  5. Después de seleccionar un reproductor, el Formulario de cliente potencial y Conexión Se mostrará la información. Confirme que esto sea correcto.
    Nota: Si se configura una experiencia de página de destino para mostrar un formulario de clientes potenciales en el cuerpo de la página, se ignorarán los ajustes de visualización y temporización del formulario de clientes potenciales.
  6. Hacer clic Ahorrar.

Asegurate que republicar la experiencia después de realizar cualquier cambio.

Exportación de datos de clientes potenciales

Todos los datos de clientes potenciales recopilados de los formularios de clientes potenciales de la Galería se exportan mediante el módulo Audiencia. Ver Exportación de datos de clientes potenciales de la galería para obtener información sobre cómo exportar los datos de clientes potenciales de la Galería.
