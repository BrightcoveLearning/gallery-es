Configuración del seguimiento de terceros para una experiencia de portal

{{ page.description }}

Para configurar los ajustes de seguimiento de terceros para una experiencia de portal, edite la experiencia y haga clic en CONFIGURACIÓN DEL SITIO> Seguimiento de terceros en la navegación de la izquierda. Las Experiencias del portal se pueden integrar o utilizar los servicios proporcionados por otros sitios de terceros:

  • Google analitico
  • Administrador de etiquetas de Google
  • Eloqua
  • Marketo

Para acceder a la configuración de seguimiento, edite la experiencia y haga clic en CONFIGURACIÓN DEL SITIO> Seguimiento de terceros en la navegación de la izquierda.

Nota: La configuración que está disponible se basa en la plantilla de sitio que se seleccionó. Si la plantilla no admite la opción, no aparecerá.

Integración con Google Analytics

La galería incluye la capacidad de enviar datos de seguimiento a Google Analytics. Cuando crea una nueva propiedad para realizar un seguimiento en Google Analytics, Google le proporcionará un ID de seguimiento y un código de seguimiento que debe agregar a su sitio. El ID de seguimiento se usa para indicarle a Google Analytics a qué cuenta y propiedad enviar datos y debe ingresarse en el campo de Google Analytics. El código de seguimiento se puede agregar a su sitio de una de estas formas:

  • HTML de encabezado / pie de página personalizado - Si está usando HTML de encabezado / pie de página personalizado para su sitio, puede pegar el código de seguimiento como parte del HTML del encabezado / pie de página
  • Especificar un archivo JavaScript personalizado: guarde el código de seguimiento en un archivo.js alojado en su dominio y, a continuación, apunte a él mediante la propiedad JavaScript personalizada del sitio

Al agregar el ID de seguimiento en la configuración del sitio, obtiene el análisis predeterminado de "vista de página". La galería agregará el JavaScript mínimo requerido para la misma. Agregar código de seguimiento como parte de un encabezado personalizado o en un archivo JavaScript personalizado le permite agregar análisis más específicos. Por ejemplo, puede agregar código personalizado para hacer clic u otros eventos.

Para obtener más información sobre Google Analytics y crear un ID de seguimiento para su sitio, visite el Google analitico sitio web.

Integración con Google Tag Manager

La galería incluye la capacidad de incluir código de Google Tag Manager. Pegue el código apropiado en el <cabeza> o <cuerpo> secciones.

Para obtener más información sobre Google Tag Manager, visite el Google sitio web.

Integrarse con Eloqua

La galería incluye scripts de seguimiento de páginas de Eloqua para permitirle realizar un seguimiento de las visitas a su sitio web. Si desea integrar su sitio con Eloqua, debe ingresar su ID de sitio Eloqua.

Integrarse con Marketo

La galería incluye secuencias de comandos de seguimiento de páginas de Marketo para permitirle realizar un seguimiento de las visitas a sus sitios de videos. Si desea integrar su sitio con Marketo, debe ingresar su ID de cuenta de Marketo.
