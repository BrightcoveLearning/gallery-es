Personalización de la configuración del formulario de reproductor y cliente potencial para una experiencia de portal

{{ page.description }}

Para personalizar la configuración del formulario de reproductor y cliente potencial, edite la experiencia del portal y haga clic en VIDEO Y REPRODUCCIÓN> Formulario de reproductor y cliente potencial en la navegación de la izquierda. La configuración del formulario de reproductor y cliente potencial permite configurar lo siguiente para una experiencia de portal:

  • reproductor
  • Comportamiento del reproductor
  • Conexión y formulario de cliente potencial

Asegúrese de volver a publicar la experiencia después de realizar cualquier cambio en la configuración del formulario del reproductor y del cliente potencial.

Usar un reproductor específico del sitio

De forma predeterminada, todas las Experiencias del portal utilizarán el reproductor predeterminado de toda la cuenta que se seleccionó como parte del Configuración de la galería. Para utilizar un reproductor específico del sitio, seleccione Usar reproductor específico del sitio y luego seleccione un reproductor para usar. Los reproductores habilitados para audiencia se indican con el icono de audiencia (). Estos reproductores están configurados con el complemento Audience y pueden realizar un seguimiento de los datos de participación en el video. Se debe usar un reproductor habilitado para público para usar un formulario de cliente potencial. Hacer clic Reproductor de vista previa para ver una vista previa del reproductor. Al pasar el cursor sobre el ícono de Audiencia, se mostrará la información de conexión de Audiencia.

Para obtener información sobre cómo crear reproductores habilitados para público, consulte Creación de un reproductor habilitado para la audiencia.

Configurar el comportamiento del reproductor

Se pueden configurar los siguientes ajustes de comportamiento del reproductor:

  • Video de reproducción automática - Si está marcado, los videos en la página de detalles se reproducirán automáticamente. Esta configuración anula la Reproducción automática de video al cargar el reproductor ajuste que está configurado en el módulo de reproductores. Esta configuración solo está disponible para las plantillas Catalog, Marquee, Publisher, Chronicle y Classic.
  • Una vez que el video termina de reproducirse, reproduce automáticamente el siguiente video de la colección - Si está seleccionada, el siguiente video de una colección comenzará a reproducirse cuando finalice un video.
  • Habilite la reproducción en línea en dispositivos iOS - Si está marcado, habilita la reproducción en línea en dispositivos iOS cuando se usa Brightcove Player versión 5.10.5 o más reciente. Esta configuración agrega el playsinline atributo al código de publicación del reproductor.

Usando un formulario de cliente potencial

Los formularios de clientes potenciales se pueden utilizar para capturar información del espectador durante la reproducción de video. Para obtener información sobre el uso de un formulario de clientes potenciales, consulte Uso de un formulario de cliente potencial con una experiencia de portal. Cuando se selecciona un reproductor habilitado para la audiencia, se mostrará el formulario de cliente potencial que se está utilizando (si corresponde) y la información de conexión de la audiencia.
