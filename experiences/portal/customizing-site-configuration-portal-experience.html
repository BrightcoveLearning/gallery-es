Personalización de la configuración del sitio de una experiencia de portal

{{ page.description }}

Para configurar los ajustes de configuración del sitio para una experiencia de portal, edite la experiencia y haga clic en CONFIGURACIÓN DEL SITIO en la navegación de la izquierda. Están disponibles las siguientes configuraciones:

  • URL - Configurar la experiencia para usar un dominio personalizado
  • Optimización de motores de búsqueda - Configure los ajustes para ayudar a mejorar el SEO
  • Control de acceso - Configurar el inicio de sesión único y las restricciones de IP
  • Seguimiento de terceros - Configurar la integración con Google, Eloqua y Marketo

Usando un dominio personalizado

El uso de un dominio personalizado le permite alojar una experiencia de portal en su propia URL. Para obtener información sobre el uso de un dominio personalizado, consulte Asignar un dominio personalizado a una experiencia de portal.

Configurar los ajustes de SEO

La configuración de Optimización del motor de búsqueda se utiliza para configurar la imagen del sitio, las palabras clave y las metaetiquetas y para acceder al mapa del sitio y a las URL del feed de búsqueda. Para obtener información sobre cómo configurar la optimización del motor de búsqueda, consulte Configurar las opciones de optimización del motor de búsqueda para una experiencia de portal.

Configurar los ajustes de control de acceso

La configuración de Control de acceso se utiliza para controlar quién puede acceder a una Experiencia de portal. Gallery puede restringir el acceso mediante un código de acceso simple, integrarse con soluciones de inicio de sesión único y puede restringir el acceso a sitios según la dirección IP del espectador. Para obtener información sobre cómo controlar el acceso a los sitios de la Galería, consulte Controlar el acceso a una experiencia de portal.

Configurar los ajustes de seguimiento

Las Experiencias del portal se pueden integrar con, o utilizar los servicios proporcionados por, otros sitios de terceros, tales como:

  • Google analitico
  • Administrador de etiquetas de Google
  • Eloqua
  • Marketo

Para obtener información sobre la integración con sitios de terceros, consulte Configuración del seguimiento de terceros para una experiencia de portal.
