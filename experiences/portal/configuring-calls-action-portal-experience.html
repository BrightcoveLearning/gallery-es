Configuración de llamadas a la acción en una experiencia de portal

{{ page.description }}

Dependiendo de la plantilla utilizada para crear la experiencia del portal, algunas de las plantillas proporcionan áreas designadas para el contenido de las llamadas a la acción. Esto puede incluir HTML, imágenes o anuncios personalizados. En las plantillas, estas áreas pueden estar etiquetadas Bandera, CTA / anuncio de cuadrícula de video, CTA / anuncio de héroe, CTA / anuncio de categoría o Anuncio / CTA de video y están designados con un color de fondo gris claro. Para configurar llamadas a la acción para una página, haga clic en un área de llamada a la acción en una página o haga clic en LLAMADOS A LA ACCIÓN en la barra de navegación izquierda y luego seleccione una plantilla de página. Se abrirá la página de Llamados a la acción.

Tenga en cuenta que la página de Llamados a la acción diferirá según la plantilla que se seleccionó. En el cuadro de diálogo anterior, se utilizó la plantilla Showcase al crear el sitio. La plantilla de la página de inicio admite llamadas a la acción en las siguientes áreas:

  • A - Encabezado de página
  • B - En la mitad o al final de la transmisión de video usando CTA repetibles
  • C - Cuando se abre la superposición para reproducir un video
  • D - En dispositivos móviles, debajo de la navegación del sitio
  • E - En dispositivos móviles, en el medio o al final de la transmisión de vídeo mediante CTA repetibles

Para configurar un llamado a la acción, haga clic en la región correspondiente y luego seleccione el tipo de acción para el área:

  • Ninguno
  • Anuncio publicitario
  • HTML personalizado
  • Imagen

Configurar llamadas a la acción repetibles

Al configurar llamadas a la acción, algunas de las áreas designadas admitían llamadas a la acción repetibles.

Para configurar la región de CTA repetible, selecciónela y luego seleccione cuándo debe mostrarse la CTA.

A continuación, seleccione el tipo de objeto que desea mostrar en el spot, Publicidad, HTML personalizado o Imagen. Para agregar espacios adicionales, haga clic en Agregar otro lugar repetido.

Configurar un anuncio

Para configurar un anuncio, seleccione Anuncio publicitario. El tamaño sugerido para el anuncio cambiará según el área seleccionada. Pegar en el Código de anuncio del encabezado del documento y el Código del bloque de anuncios.

Hacer clic Ahorrar para guardar los cambios.

Para las páginas en las que se puede reproducir un solo vídeo (páginas de detalles en directo y vídeo), si elige Publicidad, hay dos opciones de personalización adicionales disponibles:

  • Campos Personalizados - Puede optar por mostrar el código del bloque de anuncios según un valor de campo personalizado. Por ejemplo, un video con un valor personalizado de "fútbol" mostrará anuncios de fútbol. Los videos sin un valor de campo personalizado mostrarán anuncios usando el De lo contrario código del bloque de anuncios.
  • Habilitar anuncios complementarios - Si el servidor de anuncios está configurado para proporcionar anuncios complementarios, estos se mostrarán en el área de llamada a la acción cuando la reproducción de un video active el anuncio complementario.
    Nota: Para mostrar un anuncio de copia de seguridad cuando no se cumplen los criterios de campo personalizado ni los anuncios complementarios, debe proporcionar el código de bloque de anuncios en De lo contrario, mostrar; de lo contrario, se mostrará un lugar vacío.

Configurar HTML personalizado

Para configurar HTML personalizado para que se muestre, seleccione HTML personalizado. El tamaño sugerido para el HTML cambiará según el área seleccionada. Pegue el HTML personalizado que se mostrará.

Hacer clic Ahorrar para guardar los cambios.

Al configurar HTML personalizado, dependiendo de la plantilla y la región que se configure, puede haber una opción adicional para personalizar la CTA que se muestra. Por ejemplo, al configurar el CTA de superposición de la plantilla Showcase, existe una opción para:

  • Muestre el mismo llamado a la acción para todos los videos - Se usa la misma CTA para todos los videos
  • Muestre un llamado a la acción basado en valores personalizados - El CTA puede cambiar según los metadatos personalizados asociados con el video.

Tenga en cuenta que también se pueden agregar varias configuraciones.

Configurando una imagen

Para configurar una imagen para que se muestre, seleccione Imagen. El tamaño sugerido para la imagen cambiará según el área seleccionada. Arrastre y suelte una imagen del sistema de archivos o haga clic en Elija el archivo y busque y seleccione una imagen para cargar. Para agregar un hipervínculo a la imagen, proporcione una URL en el Vincular imagen a campo.

Hacer clic Ahorrar para guardar los cambios.
