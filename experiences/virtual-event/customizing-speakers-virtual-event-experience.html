    Personalización de los oradores del evento

    En este tema aprenderá a personalizar los altavoces del evento que aparecen en la página principal de Experiencia de evento virtual.

    Las imágenes de los altavoces y una breve biografía se pueden agregar a la página principal del Evento Virtual.

    La información del altavoz se personaliza editando el speakers objeto en el archivo de configuración.

    • name - Nombre del orador
    • description - Información del altavoz que se muestra debajo de su imagen. Esto se limita a aproximadamente 3 líneas de 15 palabras.
    • image_url - Imagen del altavoz. Recomienda un ancho máximo de 200 px. Las imágenes no se cambiarán de tamaño
    • link_url - (opcional) URL de destino para la imagen (puede enlazar a la página de LinkedIn del altavoz, por ejemplo)

    Por defecto, la plantilla CSS configura las imágenes del altavoz con un border-radius 50% lo que hará que las imágenes circulares.

    El speakers objeto del archivo de configuración también tiene parámetros para controlar dónde en la página de inicio se muestra la información del altavoz. Para obtener información sobre cómo colocar la sección de altavoces, consulte Posicionamiento del contenido en la página principal.
