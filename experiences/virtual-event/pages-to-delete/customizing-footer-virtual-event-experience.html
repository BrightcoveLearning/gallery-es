Personalización del pie de página de una experiencia de evento virtual

En este tema, aprenderá a personalizar el pie de página de las páginas de Virtual Event Experience.

El pie de página aparecerá en la parte inferior de todas las páginas. El pie de página se construye a partir de los valores del archivo de configuración. Cada elemento de pie de página en el archivo de configuración tendrá asociado un:

  • URL del enlace
  • Etiqueta
  • Grupo

Los grupos se utilizan para controlar dónde aparece la etiqueta / enlace en el pie de página. Hay cuatro grupos disponibles:

  • Grupo 0 - Utilizado para llamadas HTML
  • Grupo 1 - Se utiliza para texto legal e información de derechos de autor.
  • Grupos 2 y 3 - Se utiliza para listas de enlaces (se recomienda utilizar un máximo de 5 enlaces)

Los iconos sociales que aparecen en el pie de página se pueden editar / eliminar editando el social objeto en el archivo de configuración. Para obtener información, consulte Configuración de los iconos sociales para una experiencia de evento virtual.

Agregar enlaces al pie de página

Los enlaces de pie de página se configuran editando el footer_links objeto en el archivo de configuración.

  • link_url- URL de destino
  • label- Etiqueta para aparecer en el pie de página.
  • group- Grupo en el que aparece el enlace
