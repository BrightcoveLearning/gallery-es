Ejemplos de estilos de eventos virtuales

Este tema proporciona un código de muestra que se puede utilizar para personalizar la plantilla de evento virtual.

Iconos de redes sociales

Por defecto, si el configuración para compartir videos están habilitados para la experiencia, no aparecerán junto a los videos en las páginas de detalles de los videos. El siguiente CSS mostrará los íconos sociales en las páginas de detalles del video.


.share-list {
  display: block !important;
}
.share-list a {
  display: inline-block !important;
}
.share-list li {
  display: block !important;
}
