Carga de un archivo de altavoces

En este tema, aprenderá cómo cargar una lista externa de oradores para completar inicialmente la lista de oradores para una experiencia de evento virtual.

Es posible que haya creado una lista de oradores en Microsoft Excel u otra aplicación de hoja de cálculo. El módulo de agenda permite cargar un archivo de altavoces .csv externo para que todos los altavoces no tengan que introducirse manualmente en el módulo de agenda.

Nota: Brightcove recomienda cargar el archivo de altavoces antes de cargar un archivo de agenda externo.

Importar requisitos de archivo

Para importar su lista de oradores al módulo de la agenda, la lista debería haberse creado en una aplicación de hoja de cálculo (es decir, Microsoft Excel, Apple Numbers o Google Sheets). Estas aplicaciones permiten exportar los datos a un archivo.csv. El módulo de agenda solo puede importar un archivo.csv.

El módulo de agenda busca la existencia de ciertos nombres de columna. Estas columnas se asignarán a los campos correspondientes en el módulo de la agenda. Como mínimo, deben existir los siguientes campos en el archivo.csv de los altavoces:

  • last-name- Nombre del orador
  • first-name- Apellido del orador
  • email- Dirección de correo electrónico del orador

Las siguientes columnas son opcionales, pero si existen en el archivo.csv, también se importarán al módulo de agenda:

  • company- Empresa de altavoces
  • title- Título del orador
  • bio- Biografía del orador
  • social-url- URL social del orador (es decir, URL de LinkedIn)
  • homepage-url- Sitio web personal del orador
