Agregar el archivo de configuración a la experiencia

{{ page.description }}

Virtual Event Experience requiere que se incluya un archivo de configuración con el sitio. El archivo de configuración incluye la configuración del sitio, las opciones de diseño, el programa de eventos, los oradores y otro contenido que se mostrará. Después de crear una nueva experiencia de evento virtual, lo primero que se debe hacer es agregar el archivo de configuración

Para agregar el archivo de configuración a una experiencia de evento virtual, siga estos pasos.

  1. Vaya al siguiente enlace:
    https://virtual-events-experience.s3.amazonaws.com/v1/config.txt
  2. Copie el contenido al portapapeles.
  3. Abra el módulo Galería.
  4. Edita la experiencia.
  5. Hacer clic APARIENCIA Y COMPORTAMIENTO> Encabezado y pie de página en la navegación de la izquierda.
  6. Pegue el archivo de configuración en el HTML de encabezado personalizado campo.
  7. Haz clic en Guardar.

Asegúrese de volver a publicar la experiencia para que aparezcan los cambios.

Actualizando el policyKey y assetBaseUrl valores

Para que el temporizador de cuenta regresiva se muestre en videos simultáneos, policyKey El valor en el archivo de configuración debe actualizarse con una clave de política habilitada para búsqueda. El valor que se proporciona no funcionará para su cuenta. Si no lo hace, no se mostrará una cuenta regresiva con los videos. Para conocer los pasos para obtener una clave de política habilitada para búsqueda, consulte el documento para desarrolladores Ejemplo de código: Generar una clave de política. Una vez creada la nueva clave de política, actualice la policyKey valor en el archivo de configuración.

los assetBaseUrl La ruta de valor se usa para cargar archivos usados por la plantilla. Esto se puede dejar solo en la mayoría de los casos, a menos que se esté realizando un alto grado de personalización. El valor de la ruta que está allí debe eliminarse.

Después de la policyKey y assetBaseUrl Los valores se han actualizado en el archivo de configuración, vuelva a publicar la experiencia.
