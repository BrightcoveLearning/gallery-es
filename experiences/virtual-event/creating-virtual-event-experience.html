Crear una experiencia de evento virtual

{{ page.description }}

Gallery proporciona una plantilla de evento virtual que se puede utilizar para crear experiencias de eventos virtuales para albergar y realizar eventos virtuales. Después de crear una nueva experiencia de evento virtual, se le pedirá que complete previamente la experiencia. Brightcove recomienda elegir No, gracias.

El proceso para agregar videos a la experiencia es similar a otras Experiencias del Portal de Galería, aunque la plantilla de Evento Virtual espera que las colecciones se organicen de una manera específica. Para obtener información, consulte Agregar videos a una experiencia de evento virtual.

Agregar el archivo de configuración a la plantilla

Virtual Event Portal Experience requiere que se incluya un archivo de configuración con el sitio. El archivo de configuración se agrega a la plantilla como parte del encabezado de la experiencia y controla algunas de las opciones generales de la plantilla y se usa para configurar y personalizar varias secciones del sitio. Brightcove recomienda agregar el archivo de configuración inmediatamente después de crear una nueva experiencia de portal de eventos virtuales. Para obtener información, consulte Agregar el archivo de configuración a la experiencia.

Personalizando el diseño

El Editor del sitio de la galería se utiliza para personalizar el diseño de las páginas. Un logotipo, enlaces de menú, mensaje de bienvenida y el pie de página se pueden personalizar utilizando el Editor del sitio de la galería. Para obtener más información sobre cómo personalizar el diseño, consulte Personalización del contenido de las páginas de experiencia de eventos virtuales.

Configurar la agenda del evento

La agenda del evento enumera el calendario de eventos en vivo y otras sesiones. Cuando se haya creado y publicado una agenda, aparecerá en la página de inicio del evento. Para obtener información sobre cómo crear la agenda del evento, consulte Gestión de la agenda del evento.

Gestión de los ponentes del evento

La sección Oradores se completa automáticamente con todos los oradores del evento. Todos los oradores de la sesión aparecerán en la sección Oradores. Para obtener información sobre cómo administrar los oradores del evento, consulte Gestión de oradores de eventos.

Personalización de los patrocinadores del evento

La sección de patrocinadores se utiliza para destacar a los patrocinadores del evento. Para obtener más información sobre cómo personalizar los patrocinadores del evento, consulte Personalización de los patrocinadores del evento.
