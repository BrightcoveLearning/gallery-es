Especificaciones de la plantilla de evento virtual

{{ page.description }}

Video y audio

Las recomendaciones de video y audio se enumeran a continuación.

Video

Formato MP4
Códec H.264, H.265
Bitrate 2x la tasa de bits de la interpretación más alta
Cuadros por segundo Constante, generalmente entre 15 y 60 cuadros por segundo
Relación de aspecto Se conservará la relación de aspecto de su video. Es común una relación de aspecto de 16: 9.

Audio

Códec AAC (Códec de audio avanzado)
Tasa de bits Constante, 128 kbps
Canales Estéreo

Para obtener recomendaciones sobre las mejores prácticas para grabar, exportar, cargar y publicar sus videos para garantizar una reproducción óptima, consulte Mejores prácticas: El flujo de trabajo de creación, transcodificación y carga de videos.

Imagenes

Se utilizan varias imágenes en toda la plantilla de evento virtual. Las recomendaciones de imágenes se enumeran a continuación.

Imagen de fondo

El tamaño de la imagen depende de los puntos de interrupción que el cliente está tratando de admitir, la mayoría de las veces funciona con 1920 x 1080, pero la imagen debe optimizarse para que no sea gigantesca para obtener el mejor rendimiento.

El logotipo de cabecera

La imagen del logotipo del encabezado se reducirá proporcionalmente a 178 píxeles de ancho.

Agenda

El tamaño de imagen predeterminado es 320px x 180px. Las imágenes se recortarán a este tamaño.

Altavoces

Las imágenes de los oradores deben tener un ancho máximo de 200 px. Las imágenes no cambiarán de tamaño. De forma predeterminada, la plantilla CSS configura las imágenes de los oradores con un radio de borde del 50%, lo que hará que las imágenes sean circulares. La biografía del orador que se muestra debajo del orador está limitada a aproximadamente 3 líneas de 15 palabras.

Patrocinadores

Las imágenes del patrocinador cambiarán de tamaño a 100px de ancho y tendrán una opacidad establecida en 80%. CSS de muestra se proporciona para anular este comportamiento.

Fuentes

La plantilla está optimizada para trabajar con Roboto condensado fuente. Esta fuente se puede asignar usando el Estilo menú dentro de la interfaz de la Galería. Se pueden utilizar otras fuentes, pero es posible que se requieran cambios en los archivos CSS. Para obtener información sobre el uso de fuentes personalizadas, consulte Uso de fuentes personalizadas en una experiencia de portal.

Agenda

La agenda del evento enumera el calendario de eventos en vivo y otras sesiones. Cuando se haya creado y publicado una agenda, aparecerá en la página de inicio del evento. Para obtener información sobre cómo crear la agenda del evento, consulte Gestión de la agenda del evento.
