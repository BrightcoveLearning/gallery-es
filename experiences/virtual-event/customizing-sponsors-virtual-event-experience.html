Personalización de los patrocinadores del evento

{{ page.description }}

Las imágenes de los patrocinadores se pueden agregar a la página de inicio del evento virtual.

La información del patrocinador se personaliza editando el sponsors objeto en el archivo de configuración.

  • name- Nombre del patrocinador
  • image_url- Logotipo del patrocinador, las imágenes cambiarán de tamaño a 100px de ancho
  • link_url- URL de destino de la imagen

La sponsors El objeto en el archivo de configuración también tiene parámetros para controlar en qué lugar de la página de inicio se muestra la información del patrocinador. Para obtener información sobre el posicionamiento de la sección de patrocinadores, consulte Colocación de contenido en la página de inicio.

Eliminación de opacidad y tamaño de imágenes de logotipos

De forma predeterminada, las imágenes de logotipos tienen un estilo con un nivel de opacidad del 80%. El siguiente estilo CSS se puede utilizar para establecer la opacidad al 100%. La anchura máxima La propiedad se puede utilizar para cambiar el tamaño de las imágenes.


<style>
.bc-sponsors-section .bc-sponsors-list img {
    -webkit-filter: none;
    opacity: 1;
    max-width: 200;
}
</style>
Edite la experiencia y agregue el CSS en la parte superior de la HTML de encabezado personalizado . campo.
