Colocación de contenido en la página de inicio

{{ page.description }}

Varios objetos en el archivo de configuración tienen parámetros que le permiten especificar dónde debe aparecer el objeto en la página. Por ejemplo, el contenido estático, los altavoces, las pistas y los patrocinadores tienen la capacidad de reposicionarse en la página. Los objetos que se pueden posicionar tendrán los siguientes atributos en el archivo de configuración:

  • insert_target- Un selector de CSS (una clase o ID simple) que se utilizará como punto de partida para el procesamiento del objeto dado
  • insert_mode- Cómo se inserta el contenido
    • after
    • before
    • append
    • prepend

Determinando el objetivo

La insert_target El parámetro espera un selector de CSS (usando una clase simple o formato de ID). El elemento que coincide con este selector se utilizará como punto de partida para el procesamiento del objeto dado. La mejor manera de determinar el selector o ID de CSS que se debe usar es navegar a la página de inicio del evento virtual y luego usar las herramientas de desarrollo del navegador para inspeccionar los elementos en la página.

Notará que las secciones principales de la página de inicio están organizadas usando <div> etiquetas.

Elegir el modo de inserción

Después de determinar en qué lugar de la jerarquía insertar el objeto, el insert_mode El atributo determinará cómo se inserta el objeto en la jerarquía.

  • after- Pone el objeto después del especificado div al mismo nivel
  • before- Pone el objeto antes del especificado div al mismo nivel
  • append- Pone el objeto dentro del especificado div como el ultimo hijo
  • prepend- Pone el objeto dentro del especificado div como el primer hijo

El diagrama siguiente demuestra el efecto de los distintos insert_mode valores.

Ejemplo de posicionamiento de objetos

Los siguientes ejemplos de código demuestran cómo usar un objeto de contenido estático para insertar un título encima de la colección secundaria.


{
    "insert_target": ".bc-on-demand-section",
    "insert_mode": "prepend",
    "content": "<h2 class='bc-on-demand-section-title'>ON-DEMAND SESIONS</h2>",
    "require_roles": [],
    "omit_roles": []
}

Con las herramientas del navegador, puede ver el h2 fue insertado como el primer elemento dentro de la bc-on-demand-section div.
