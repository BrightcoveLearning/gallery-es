Agregar videos a una experiencia de evento virtual

{{ page.description }}

Con el Editor de colección de galería, puede seleccionar los videos para mostrar en una experiencia de evento virtual. Al igual que otras experiencias de tipo portal, los videos se organizan en colecciones.

Para agregar videos a una experiencia de evento virtual, haga clic en VIDEOS en la navegación de la izquierda.

Organización de la colección de videos

El orden de las colecciones controlará cómo aparecen en la página de inicio del evento virtual.

Nota: Para obtener información sobre la creación de colecciones y la adición de vídeos a una colección, consulte Adición de vídeos a una experiencia de portal.

La plantilla de evento virtual tiene cuatro áreas principales donde los videos se muestran como se indica en el sitio de muestra:

  1. Colección destacada - Normalmente se utiliza para presentar en vivo o contenido simultáneo que se mostrará durante todo el evento. Idealmente, esto debe ordenarse en función de su tiempo de activación, desde el más antiguo al más reciente, para obtener la mejor experiencia.
  2. Colección primaria - Puede incluir cualquier cosa, pero generalmente se usa para presentar desgloses de VOD específicos o contenido al que desea atraer atención adicional
  3. Colección secundaria - Puede incluir cualquier cosa, pero generalmente se usa para presentar desgloses de VOD específicos o contenido al que desea atraer atención adicional
  4. Pistas - Normalmente se utiliza para contenido VOD
Nota: Las colecciones Destacadas, Primaria y Secundaria admiten un máximo de 24 vídeos.
Nota: Para insertar un título sobre la colección primaria / secundaria, agregue un objeto de contenido estático al archivo de configuración. Para obtener información, consulte Trabajo con contenido estático.

Al ordenar colecciones en la Galería, las colecciones deben ordenarse de la siguiente manera:

  • Colección 1 - Colección destacada
  • Colección 2 - Colección primaria
  • Colección 3 - Colección secundaria
  • Colección 4 - Pistas

Configurar pistas de eventos

Al crear las pistas, siga las pautas a continuación para asegurarse de que aparezcan correctamente en la experiencia:

  • Los nombres de las colecciones aparecerán en la plantilla del evento virtual, así que use nombres significativos.
  • La colección de pistas será una colección anidada, no aparecerán videos en esta colección.
  • La colección de pistas debería ser la cuarta colección.
  • Cree colecciones secundarias en la colección de pistas para cada pista.
  • No utilice listas de reproducción al crear las colecciones.
  • La posición de la sección de pistas se especifica en el archivo de configuración y puede ser cambiado editando el breakout_collection objeto en el archivo de configuración

Configurar la URL de las pistas

Una vez que se han creado las colecciones, el archivo de configuración debe actualizarse para que el enlace de sesiones bajo demanda en el encabezado dirija a la URL correcta. El nombre utilizado para la colección de pistas (colección 4) se utilizará en la URL para acceder a la página de pistas. Para asegurarse de que la URL esté configurada correctamente, confirme lo siguiente.

  1. Revise el archivo de configuración y localice el breakout_collection objeto.
  2. La collection El parámetro contiene la URL de la página de pistas. Por ejemplo:
    
"breakout_collection": {
    "collection": "/category/videos/tracks",
    "insert_target": ".bc-on-demand-section",
    "insert_mode": "after"
},
  3. El último camino en el collection La URL debe reflejar el nombre de su colección de pistas (colección 4). Por ejemplo, si su colección de pistas (colección 4) se llama 'Sesiones', el parámetro de colección se actualizará a:
    
"collection": "/category/videos/sessions",
  4. Para verificar la configuración, haga clic en el enlace Sesiones bajo demanda en el encabezado. Debería aparecer la sesión de pistas. Hacer clic en una pista debería abrir los videos en una pista. Si ve un error 404, hay un problema de configuración.
