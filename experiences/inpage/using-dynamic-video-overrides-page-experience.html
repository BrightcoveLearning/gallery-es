Uso de anulaciones dinámicas de video en una experiencia en la página

{{ page.description }}

Puede haber ocasiones en las que desee cambiar los videos en una experiencia en la página. Por ejemplo, tal vez los videos de una experiencia deban cambiar en función de una selección que realice el usuario o en función de la navegación a otras partes de un sitio. En lugar de crear nuevas experiencias en la página, cada una con un conjunto de videos diferente, es posible modificar el código de inserción para asignar los videos apropiados a una experiencia. Los cambios en el código de inserción varían según el tipo de código de inserción que se esté utilizando:

  • Código de inserción recomendado
  • Sin código de inserción de JavaScript

Código de inserción recomendado

El código de inserción típico de la experiencia en la página (opción recomendada) tiene el siguiente aspecto:

<div data-experience="597a2aa114b2ed00103052af"></div>
<script src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/live.js"></script>

Para anular dinámicamente los videos asociados con una experiencia en la página, el código de inserción se puede cambiar para agregar un atributo al div etiqueta para especificar los videos o la lista de reproducción que se utilizará.

Especificar videos individuales

Para especificar videos individuales para usar, agregue el data-video-ids parámetro al div etiqueta y luego especifique una lista delimitada por comas de ID de video.

<div data-experience="597a2aa114b2ed00103052af" 
     data-video-ids="12345,56789"></div>
<script src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/live.js"></script>

Especificar una lista de reproducción

Para especificar una lista de reproducción para usar, agregue el data-playlist-id parámetro al div etiqueta y luego especifique el ID de la lista de reproducción que se utilizará.

<div data-experience="597a2aa114b2ed00103052af" 
     data-playlist-id="12345"></div>
<script src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/live.js"></script>

Sin código de inserción de JavaScript

El código de inserción típico de la experiencia en la página (sin opción de JavaScript) tiene el siguiente aspecto:

<iframe style="display:block;border:none;margin-left:auto;margin-right:auto;"
src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/index.html"
allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>

Para anular dinámicamente los videos asociados con una experiencia en la página, el código de inserción se puede cambiar para agregar un parámetro al src atributo.

Especificar videos individuales

Para especificar videos individuales para usar, agregue el videoIds y luego especifique una lista delimitada por comas de ID de video.

<iframe style="display:block;border:none;margin-left:auto;margin-right:auto;"
src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/index.html?videoIds=12345,56789""
allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>

Especificar una lista de reproducción

Para especificar una lista de reproducción para usar, agregue el playlistId y luego especifique el ID de la lista de reproducción que se utilizará.

<iframe style="display:block;border:none;margin-left:auto;margin-right:auto;"
src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/index.html?playlistId=12345"
allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>
