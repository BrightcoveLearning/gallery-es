Configurar los detalles para una experiencia en la página

{{ page.description }}

El nombre de la experiencia, la descripción y el tamaño del diseño se pueden configurar como parte de la configuración general. Para configurar los ajustes de descripción general, edite la experiencia y luego haga clic en INFORMACIÓN GENERAL en la navegación de la izquierda.

La página de descripción general muestra:

  • Tipo de experiencia - Muestra la plantilla que se utiliza para la experiencia in-page.
  • Título y descripción de la experiencia - Cuando crea una nueva experiencia por primera vez, se le solicita que ingrese Título de la experiencia y Descripción de la experiencia. Estos valores se pueden editar aquí.
  • Experimentar el idioma - Seleccione el idioma que se utilizará para mostrar los mensajes de la interfaz de usuario
  • Tamaño de diseño - Determina el tamaño de la experiencia in-page. Ver Establecer el tamaño de la experiencia para detalles.

Cuando termine, haga clic en Ahorrar para guardar sus cambios.

Nota: Después de realizar cambios en una experiencia, asegúrese de volver a publicar la experiencia para que los cambios aparezcan en los sitios que muestran la experiencia.

Establecer el tamaño de la experiencia

Las experiencias in-page se pueden dimensionar de tres maneras:

  • Sensible - La experiencia cambiará de tamaño automáticamente para adaptarse a su contenedor.
    Nota: Al seleccionar esta opción, se debe utilizar el código de inserción recomendado. El código de incrustación sin JavaScript no responde.
  • Tamaño fijo - El tamaño de la experiencia se dimensionará de acuerdo con el especificado. ancho y altura valores.
  • Ancho fijo -El ancho de la experiencia se establecerá al especificado ancho valor.

Cuando termine, haga clic en Ahorrar para guardar sus cambios.

Nota: Cambiar el tamaño del diseño requerirá que la experiencia se vuelva a publicar y que se actualice todo el código de publicación incrustado existente. Consulte Vista previa y publicación de una experiencia en la página para obtener más información.
