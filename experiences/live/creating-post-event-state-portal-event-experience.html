    Creación del estado posterior al evento para una experiencia de evento de portal

    En este tema aprenderá a crear el estado posterior al evento para una experiencia de portal de eventos en vivo.

    La plantilla Portal de eventos en vivo proporciona plantillas de pre-evento, evento en vivo, post-evento, archivo de vídeo y página de detalles de vídeo. Estas plantillas de página se pueden personalizar de manera que el sitio tenga una apariencia distinta antes, durante y después de cada evento en directo. El estado posterior al evento es lo que mostrará el sitio cuando se haya completado el evento en vivo.

    Para crear el estado de la página de inicio posterior al evento de un sitio, cree una nueva experiencia de portal de eventos en vivo. En el Editor del sitio de la galería, haga clic en PAGES en la navegación izquierda y, a continuación, haga clic en Página principal: Post-evento.

    El estado de la página principal posterior al evento proporciona las siguientes áreas de contenido.

    • A - Logotipo y nombre de inicio : se utiliza para devolver el visor a la página principal. Aquí aparecen dos marcadores de posición que permiten, por ejemplo, colocar una imagen seguida de un texto.
    • B - Enlace al archivo de vídeos - Permite ver todas las colecciones de vídeos creadas para el sitio.
    • C - Ajustes de Twitter - Permite utilizar widgets de feeds de Twitter. Consulte Configuración de un feed de Twitter.
    • D - Imagen de fondo, texto y enlaces - Permite añadir una imagen de fondo, texto de subtítulos abiertos y enlaces.
    • E - Nombre del evento en vivo - Mensaje de texto estático
    • F - La transmisión en vivo ha finalizado el mensaje - Mensajede texto estático
    • G - Configuración de registro de correo electrónico - Se utiliza para configurar el registro de correo electrónico. Consulte Configuración de la inscripción por correo electrónico.
    • H - Añadir vídeos - Se utiliza para añadir colecciones de vídeos al sitio. Para obtener más información, consulte Añadir vídeos.
    • I - Video Grid Box- Se utiliza para agregar llamadas a la acción, incluyendo HTML personalizado o imágenes. Consulte Configuración de llamadas a la acción en una experiencia de portal para obtener más información.
    • J - Objetos de pie de página - Permite incluir texto y enlaces en el pie de página.

    Para obtener información sobre cómo agregar imágenes, vínculos y texto a una plantilla de página de Galería, consulte Personalización del contenido de las páginas de experiencia de portal.

    Nota: Los colores del sitio pueden variar según el tema que se haya seleccionado.

    Configuración de un feed de Twitter

    El sitio se puede configurar con una inserción de Twitter en la página. Para conocer los tipos de incrustaciones disponibles, consulte el sitio web de Twitter. Una línea de tiempo de muestra le gusta se muestra a continuación.

    Para publicar un feed de Twitter, siga estos pasos:

    1. Cree un widget en el sitio de Twitter. Si necesita ayuda para crear un feed de Twitter, consulte la documentación para desarrolladores de Twitter.
    2. En la página, haga clic en el enlace Ajustes de Twitter.
    3. Copie el código HTML del widget de Twitter y péguelo en los ajustes de feeds de Twitter.
    4. Haga clic en Guardar.

    Configuración de la inscripción por correo electrónico

    El sitio se puede configurar de manera que los visitantes del sitio puedan indicar su dirección de correo electrónico.

    Para añadir una inscripción por correo electrónico, siga estos pasos:

    1. En la página, haga clic en el enlace Ajustes de inscripción por correo electrónico.
    2. Configure las propiedades de inscripción por correo electrónico que necesite.
      • No mostrar el formulario de prospecto de correo electrónico - No mostrar el formulario de correo electrónico
      • Usar el formulario de correo electrónico proporcionado con la plantilla - Seleccione esta opción para utilizar el formulario de correo electrónico predeterminado proporcionado con la plantilla
      • Usar HTML personalizado proporcionado por su software de automatización de marketing - Seleccione esta opción para utilizar el formulario personalizado HTML para la captura de correo electrónico
    3. Haga clic en Guardar.
    4. Una vez configurada la configuración del correo electrónico, vuelva a la página y haga clic en el enlace Mensaje de suscripción por correo electrónico y establezca el texto que aparece encima del campo de registro de correo electrónico.

    Todos los datos capturados por el formulario de correo electrónico se pueden exportar mediante la opción Exportar clientes potenciales de la galería en el módulo Audiencia. Los clientes potenciales capturados utilizando la opción HTML personalizada se ven dentro de su software de automatización de marketing.

    Añadir vídeos

    Para añadir colecciones de vídeos al sitio, haga clic en el enlace +Añadir vídeos. Para obtener información sobre cómo crear colecciones, consulte Agregar vídeos a una experiencia de portal.

    La primera colección de la lista de colecciones se mostrará en la página de inicio posterior al evento. Para ver todas las colecciones, haga clic en uno de los vídeos de la página principal o configure el objeto Vínculo de archivo de vídeo para que los espectadores puedan hacer clic en él para ver todas las colecciones.
