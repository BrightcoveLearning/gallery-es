Referencia de errores de la API de experiencia en la página

Este tema proporciona una referencia de error para la API de experiencia en la página (la API de la plataforma).

Errores de la API de experiencia en la página

Referencia de errores de la API de experiencia en la página
Error Ejemplos de Descripción
400 BAD_REQUEST 
{
  "status": 400,
  "code": "BAD_REQUEST",
  "message": "No template specified"
}
{
  "status": 400,
  "code": "BAD_REQUEST",
  "message": "Specified template does not exist"
}
{
  "status": 400,
  "code": "BAD_REQUEST",
  "details": [
    "Path `name` is required."
  ]
}
 Por lo general, un problema con el cuerpo de la solicitud, como omitir el template o algún otro campo obligatorio, o especificando una plantilla que no existe
401 UNAUTHORIZED 
[
  {
    "error_code": "UNAUTHORIZED",
    "message": "Permission denied."
  }
]
 No se envió ningún token de acceso con la solicitud o las credenciales del cliente utilizadas para crear el token de acceso no tenían los permisos correctos para la solicitud.
404 NOT_FOUND 
{
  "status": 404,
  "code": "NOT_FOUND",
  "message": "Experience 5bc2624e0f6f340011b not found"
}
 La experiencia que está intentando actualizar no existe. Es posible que se haya eliminado o que la identificación de experiencia en la solicitud sea incorrecta.
