Configurar los ajustes de la galería

{{ page.description }}
 

Para configurar los ajustes de la Galería, abra el módulo Galería y haga clic en el Ajustes enlace en la parte superior de la página. Se pueden configurar los siguientes ajustes:

  • Jugadores
  • YouTube
  • Dominios personalizados y SSL
  • Perfiles de control de acceso
  • HTML personalizado

Selección de un reproductor de Video Cloud predeterminado

La galería debe configurarse con un reproductor de video predeterminado para toda la cuenta que se utilizará en sus sitios de galería. Los reproductores de los sitios individuales de la Galería también se pueden configurar como parte de la configuración de VIDEO Y REPRODUCCIÓN, lo que brinda la capacidad de anular el reproductor predeterminado de toda la cuenta. Para más información, ver Personalización de la configuración del formulario de jugador y cliente potencial para una experiencia de portal.

Como práctica recomendada, le recomendamos crear y diseñar reproductores dedicados para sus sitios de Galería. Si no ha creado un reproductor dedicado para su sitio de Galería, debería comenzar por hacerlo. Para obtener información sobre la creación de reproductores, consulte Introducción al módulo de jugadores.

Para configurar la Galería con un reproductor de video predeterminado, siga estos pasos.

  1. Hacer clic Jugadores en la navegación de la izquierda.
  2. Seleccione el reproductor predeterminado de su cuenta de la lista de jugadores y luego haga clic en Ahorrar.
Nota: Debe seleccionarse un reproductor predeterminado o Portal Experiences no se publicará.

Los reproductores habilitados para audiencia se indican con el icono de audiencia (). Estos reproductores están configurados con el complemento Audience y pueden realizar un seguimiento de los datos de participación en el video. Al pasar el cursor sobre el icono, se mostrará la información de conexión de la audiencia.

Para obtener información sobre cómo crear reproductores habilitados para público, consulte Creación de un reproductor habilitado para la audiencia.

Configurar los ajustes de YouTube

Los sitios de la galería pueden contener videos de su cuenta de Video Cloud o de una cuenta de YouTube. Antes de que los videos de YouTube se puedan agregar a un sitio de la Galería, se deben configurar los ajustes de YouTube. Para obtener información sobre esta configuración, consulte Configurar los ajustes de YouTube.

Configurar dominios personalizados y SSL

Cuando crea un sitio de Galería por primera vez, se le asigna un dominio aleatorio con el formato de <site-slug>.bcovp0rtal.com. Gallery ofrece la posibilidad de publicar sitios de Gallery en un dominio proporcionado por Brightcove o utilizando su propio dominio personalizado. Los sitios también se pueden proteger mediante SSL. Para obtener información sobre cómo configurar dominios personalizados y SSL, consulte Configuración de dominios personalizados y SSL.

Configuración de perfiles de control de acceso

Se pueden crear perfiles de control de acceso para controlar el acceso a los sitios de la Galería. El acceso a un sitio puede estar limitado por:

  • Dirección IP: el acceso está limitado en función de la dirección IP del visor
  • Credenciales SSO - Se requiere un nombre de usuario y una contraseña para acceder
  • Código de acceso: el usuario debe introducir un código de acceso antes de que se muestre el sitio (útil para crear y probar un sitio)

Para obtener información sobre la creación de perfiles de control de acceso, consulte Controlar el acceso a una experiencia de portal.

Uso de HTML de encabezado / pie de página personalizado

Gallery ofrece la opción de utilizar un encabezado y / o pie de página personalizados para todos sus sitios. Esto brinda la posibilidad de personalizar los sitios de la Galería para que se ajusten más a la apariencia de sus sitios corporativos. También se puede configurar un encabezado / pie de página personalizado para un sitio individual. Para obtener información sobre cómo configurar un encabezado / pie de página personalizado para un sitio, consulte Personalización de la apariencia y el comportamiento de una experiencia de portal.

Para configurar la Galería para usar un encabezado o pie de página personalizado, comience por diseñar el encabezado / pie de página para usar en sus sitios. El encabezado / pie de página puede contener cualquier código HTML, JavaScript o CSS válido. Cuando termine, siga estos pasos.

  1. Hacer clic HTML personalizado en la navegación de la izquierda.
  2. Pegue el HTML del encabezado en el HTML de encabezado personalizado campo.
  3. Pegue el HTML del pie de página en el HTML de pie de página personalizado campo.
  4. Hacer clic Ahorrar para guardar los cambios.

Notas sobre el uso de encabezados y pies de página personalizados

  • La galería no valida el HTML que se proporciona para el encabezado / pie de página
  • No hay límite de caracteres explícito en el HTML del encabezado y pie de página.
  • Se admiten archivos CSS y JavaScript externos
  • El HTML personalizado debe seguir los principios de diseño receptivo para que se adapte correctamente a todos los dispositivos.
  • Brightcove ha visto conflictos de CSS cuando los sitios usan Bootstrap (la Galería también usa Bootstrap); asegúrese de verificar y resolver cualquier conflicto
